## üöÄ Getting Started

### Prerequisites
- Python 3.9+
- NumPy, GUDHI, Ripser libraries
- Optional: CUDA toolkit for GPU acceleration

### Installation
```bash
git clone https://github.com/quantum-fortress/quantum-fortress-2.0.git
cd quantum-fortress-2.0
pip install -r requirements.txt
```

### Basic Usage
```python
from quantum_fortress import QuantumFortress

# Initialize with default 4D hypercube
qf = QuantumFortress(dimension=4)

# Generate hybrid keys (ECDSA + quantum-topological)
keys = qf.generate_keys()

# Sign a transaction
signature = qf.sign(keys["private"], "transaction_data")

# Verify the signature
is_valid = qf.verify(keys["public"], "transaction_data", signature)

# Check topological vulnerability index
tvi = qf.get_tvi(signature)
print(f"Transaction TVI: {tvi:.4f} {'(SECURE)' if tvi < 0.5 else '(VULNERABLE)'}")
```
